Configuration Supabase

Renseignez l'URL et la cle publique de votre projet Supabase.
Ces informations sont dans Project Settings > API de votre tableau de bord Supabase.

Initialiser la base de donnees

Copiez ce SQL dans Supabase > SQL Editor > New query puis cliquez Run.

-- Table clients CREATE TABLE clients ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), nom text NOT NULL, type text, contact text, email text, telephone text, adresse text, departement text, notes text, created_at timestamptz DEFAULT now() ); -- Table contrats CREATE TABLE contrats ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), client_id uuid REFERENCES clients(id) ON DELETE CASCADE, produit text NOT NULL, detail text, periodicite text NOT NULL DEFAULT 'Annuel', montant_ht numeric(10,2), derniere_facture date, echeance_contractuelle date, facture_ebp text, notes text, actif boolean DEFAULT true, created_at timestamptz DEFAULT now() ); CREATE INDEX ON contrats(client_id); CREATE INDEX ON contrats(echeance_contractuelle);
